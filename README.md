# POR Marketplace

A Claude Code plugin marketplace for structured development workflows.

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [por-dev](./por-dev) | Structured workflows for feature development, bug fixes, and chores following the Product on Rails methodology |

## Installation

Add this marketplace to Claude Code:

```bash
claude plugin marketplace add https://github.com/lfnovo/por-marketplace
```

Then install any plugin:

```bash
claude plugin install por-dev@por-marketplace
```
